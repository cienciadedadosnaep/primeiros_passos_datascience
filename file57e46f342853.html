Primeiros Passos

Apresentação:

O projeto Ciência de Dados na Educação Pública tem desenvolvido atividades que exigem a utilização de diversas ferramentas aplicadas à área de ciência de dados. A cada semestre é realizado um pequeno treinamento que permite que os novos integrantes sejam recepicionados dentro da metodologia do projeto e ao mesmo tempo tornem-se autônomos no desenvolvimento de organização de dados, visualização e transformação em materiais didáticos. O curso é aberto a participantes de graduação e pós-graduação interessados em iniciar na área de tecnologia.

Objetivo:

Desenvolver habilidades no uso de ferramentas de consulta, manipulação e transformação de dados para produção de materiais didáticos e divulgação de dados referentes ao site (re)conhecendo Salvador.

Conteúdos

  • Git: git init,git add, git commit -m, git checkout, git merge, git status, git log, git pull, git pull request, git push, git fetch.
  • Github: Criação de conta do Github
  • Instalação do R e Rstudio
  • Instalação Vscode
  • Configuração da conta do Github no Rstudio e Vscode
  • MySQL: SHOW DATABASES, USE, DESCRIBE, INSERT INTO, SELECT, FROM, WHERE
  • Instalação MySQL e Workbench
  • Conexão Banco: R e Python
  • Tidyverse: readr, dplyr (select(),filter(),group_by(),summarise(),mutate()), tidyr (gather(),spread()), lubridate, ggplot2 (geom_bar(),geom_col())
  • Python: Vscode/Colab/Jupyter,**Pandas,Numpy,read_csv(…,chuncksize=…)*
