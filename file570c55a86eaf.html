Git e GitHub

O primeiro passo para utilizar o Git é realizar sua instalação por meio do link Download Git.

  1. git init
  2. git add codigo.R
  3. git commit -m “definicao do arquivo”
  4. git log

alterar arquivo -codigo.R- incluindo algum cálculo.

  1. git add codigo.R
  2. git commit -m “update”
  3. git status
  4. git log
  5. git show colocar o número do commit

linhas paralelas

  1. git checkout feature/funcao

criar função no arquivo codigo.R

  1. git add codigo.R
  2. git commit -m “inclusao da funcao”
  3. git checkout master

com o comando 13. retornamos para a branch master.

  1. git merge feature/funcao
